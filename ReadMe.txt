_Steps To Install the System_
Download and Install this Component 

Git
VsCode
Composer
Xampp
NodeJs 

watch this video for Laravel installation
https://www.youtube.com/watch?v=aujB-TLZCDE

Follow these steps for breeze installation
composer require laravel/breeze --dev

php artisan breeze:install (choose these options= blade, yes or no, Pest)

php artisan migrate
npm install
npm run dev


After creating your project
replace the project you created with the Voting system folder
then replace your database with this vote3 database
